GIT NOTES

    git = software for saving code

    -go to dir from terminal
    -initialize folder -> in dir pass 'git init' (inserting memory card)
    -save progress ->pass 'git add' + :
        'filename'(save specific file)
        '.'(save everything)

    -after passing git add to save, 
    commit changes with 'git commit -m'+ 'your message'
    EX:
        git commit -m 'saved python file'

    -look at saves with 'git log'(log of saved changes)
    EX:
        commit 78b8f228849f523525a8eb997cebb2bd67996807 (HEAD -> main)
        Author: Nolan A Milligan <noamilligan@Nolans-MAC.local>
        Date:   Sat Oct 18 23:46:15 2025 -0400

            saved NOLAN CODE dir

    -to go back -> copy the Hash (78b8f228849f523525a8eb997cebb2bd67996807)
    pass 'git checkout <hash>'
    goes to old branch of that hash save

    -pass 'git branch' to view current branch
